FMDe3

SOAP Version: 1.2

Main class which you can use to connect to your web service

Important settings

  • Url - the web service address. By default, it is retrieved from the wsdl file
    Example:
                        
FMDe3 service = new FMDe3("https://wyszukiwarkaregontest.stat.gov.pl/wsBIR/UslugaBIRzewnPubl.svc");
service.GetValue();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
FMDe3 service = new FMDe3();
service.getHttpHeaders().put("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l");
service.GetValue();
  • Enable logging - if you enable logging, all requests/responses will be written to the console (for debugging purposes)
    Example:
                        
FMDe3 service = new FMDe3();
service.setLoggingEnabled(true);
service.GetValue();
